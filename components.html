Bootstrap Components and Classes Used

The site uses Bootstrap 5 for layout, grid, and utility classes, while all visual styling and interactivity are implemented manually.

  • Grid system: .container, .row, .col-lg-* to layout the hero, education cards, experience, and other sections.
  • Cards: Education and experience details use .card, .card-body, and .border-secondary-subtle as a base.
  • Navigation (pills): The side navigation uses .nav, .flex-column, and .nav-pills for the vertical navigation structure.
  • Typography utilities: Classes such as .display-3, .lead, .h5, .small, and .text-secondary for consistent headings and text.
  • Spacing utilities: Margin and padding helpers like .mb-3, .mt-4, .py-3, .g-4 to control spacing between sections and components.
  • Flex utilities: .d-flex, .align-items-center, .justify-content-between for aligning content inside cards, the hero section, and the experience layout.
  • Responsive behaviour: Bootstrap’s responsive breakpoints are used with .col-lg-* and media queries in custom CSS to ensure usability on mobile, tablet, and desktop.

Custom JavaScript Features

All dynamic behaviour required by the CA brief is implemented with custom JavaScript in js/main.js; it does not rely on Bootstrap’s JavaScript plugins.

  • Theme switcher: A “theme toggle” button applies and removes a .theme-light class on the <body>, switching between dark and light glass themes.
  • Smooth scrolling navigation: Clicking side navigation links scrolls smoothly to the chosen section and briefly highlights the active nav item.
  • Contact modal dialog: The “Contact” nav item and “Contact me” hero button open a custom glass overlay with a contact form. The overlay can be closed by clicking outside the panel or pressing Esc.
  • Email sending via form: The “Send” button reads the form fields and opens the user’s default mail client using a mailto: link that pre-populates the subject and body with the entered details.
  • Animated background blobs: Three blurred “blob” elements in the background respond to mouse movement, creating a subtle parallax effect.
  • Tech Stack Meter (skill bars): Custom IntersectionObserver logic fills glass “skill bars” once they scroll into view. Each bar animates from 0 % width to its configured percentage and adds a soft glow.
  • Skill bar hover ripple: Moving the cursor over a skill bar updates CSS custom properties to draw a radial highlight under the pointer, creating a ripple effect on hover.
  • Hobbies glass cards: Hobbies/Interests are displayed in transparent glass cards that lift and glow slightly on hover for a subtle interactive feel.
